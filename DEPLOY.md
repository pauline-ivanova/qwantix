# Инструкция по публикации сайта

## Способ 1: Через веб-интерфейс Vercel (Рекомендуется)

1. Перейдите на https://vercel.com
2. Войдите в аккаунт (или создайте новый)
3. Нажмите "Add New Project"
4. Выберите репозиторий: `pauline-ivanova/qwantix`
5. Vercel автоматически определит Next.js и настроит проект
6. Нажмите "Deploy"
7. После деплоя вы получите URL вида: `https://qwantix-agency.vercel.app`

## Способ 2: Через Vercel CLI

1. Выполните в терминале:
   ```powershell
   vercel login
   ```
2. Откройте ссылку, которая появится в терминале
3. Авторизуйтесь в браузере
4. После авторизации выполните:
   ```powershell
   vercel --prod --yes
   ```
